webapis小细节

  • 2024-05-14
  • 前端
易错点和注意事项

  1. 到底使用的是querySelector 还是 querySelectorAll ，关键看获取的是一个还是多个

  2. 到底要不要写for循环，看操作的是不是元素集合伪数组，如果是元素集合伪数组，就需要for循环遍历操作处理。

  3. 伪数组，也有数字下标，也是从0开始的。

  4. 要不要使用${}

    1. ${} 的使用是在 反引号里面来使用的，反引号表示是个字符串
    2. 所以不涉及到反引号，不涉及字符串的化，就不要书写 ${} 了

  5. 是属性还是方法

    1. 属性不需要括号

    2. 方法是需要的

  6. 到底要不要加引号

    1. 我们操作的值如果是字符串，就需要带引号

    2. 如果非字符串，就可以不用引号

  7. 给元素注册事件的时候，到底要不要使用事件委托呢

    1. 当给多个元素注册事件的时候，可以使用事件委托来优化
    2. 并且当给动态生成的元素来注册事件的时候，使用事件委托来实现。

  8. 操作内容，到底是通过 innerHTML 还是 value 属性

    1. 对于像 div span ul li p 等标签，操作内容使用 innerHTML 或 innerText 来实现
    2. 但是对于像表单元素 input、单选、多选、文本域等，操作内容就需要通过 value 属性来操作了
    3. 特殊的：button按钮的内容是需要通过 innerText 或 innerHTML 来设置了

  9. 设置样式的时候，是否需要单位px

    1. 对于style设置样式，大部分需要带单位px
    2. 比如fontSize、width 、height
    3. 但对于层级z-index ，背景系列background不需要
    4. 另外三大家族的scrollTop的值是不需要单位px的
